Modulo Circle p5.js

IFS-Plotting

This is a project that uses Python loops to repeatedly iterate inputs through an iterated function system, or an IFS. I used numpy and basic python functions to perform the computations and matplotlib to plot the results, which are fractals that depend on the IFS. The post tries to explain and teach some of the math involved.

Research (ML/Data)

IN PROGRESS: I am planning on uploading a Jupyter notebook file to demonstrate the logic. In this research, I extracted data from an inertial measurement unit (IMU) and then performed some data cleaning (filtering, smoothing, transformation/reorientation). We then trained a supervised, k-nearest-neighbors ML model to classify various activities of the wearer of the IMU.

Trig Animation p5.js

This is a very simple animation demonstrating how the sine of a function is just the height of a radius, revolved by some angle about the center. Similarly, the cosine is the horizontal displacement after moving some angle around the circle.
